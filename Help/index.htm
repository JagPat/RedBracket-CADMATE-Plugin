1. Introduction

 

AutoCAD redbracket connector is an add-on tool for AutoCAD to seamlessly work with redbracket system directly from AutoCAD. This add-on automates several processes of redbracket’s drawing management like versioning, x-ref, layouts and title blocks directly from AutoCAD. This helps in increased user productivity as user need not be well versed with redbracket and its user interface but still can manage the data in redbracket.

Installation Procedure

 

To install RedBracket Connector, do as follows:

 

  1. Ensure that your computer meets the minimum System Requirements.
  2. Ensure that you have installed AutoCAD 2013 onwards.
  3. Double click Redbracket-AutoCAD.msi.
  4. Follow the instructions in the Setup Wizard.
  5. Open AutoCAD and start working.

3.4. Lock / Unlock

Using this command, you can lock/unlock the current open file on redbracket.

 

 

 

 

 

Clicking on Lock/Un-lock will open below screen.

 

 

 

 

Click on "Lock" to lock file and then same button will be renamed to "Unlock", you can Unlock file by clicking on "Unlock" and vice-versa.

Typically you can lock the file to get exclusive write access to the document when you want to edit it. If some other user has already acquired lock on the file, then the lock operation would fail and you need to wait till the time other user releases the lock.

Once you acquire lock, you can edit the document. After you finish your work and save file in redbracket, system will unlock the file and releases exclusive write access to the document so that other users can edit the file and make new versions.

3.9. Compare Drawing Info and Refresh

Current version and Latest version of Currently opened drawing will be displayed here, if current version is different from latest version then it will be blinking with "is Old".

 

By clicking on Refresh button Drawing in case when version and last updated time of both drawing is same message-box will popup saying Drawing is latest.In case when Version are same and Last updated time is different Local file properties will be updated with server file properties. In case when versions are different then message-box will pop saying "Do you want to download latest file from server?" Its entirly up to user to decide whether to download the latest file or not, if user decides to download latest file then current file will be lost.
